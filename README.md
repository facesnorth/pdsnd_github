# Project Title
[pdsnd_github] or **Let's Explore Some Bikeshare Data!**

## Description
This project examines some statistical analysis of bike share data from 3 cities: New York, Chicago, and Washington.

### Files used

- bikeshare.py
- chicago.csv
- new_york_city.csv
- washington.csv
